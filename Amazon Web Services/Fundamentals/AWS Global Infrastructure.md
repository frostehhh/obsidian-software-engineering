# AWS Regions
Creations of AWS that they have selected an inside this region is a full deployment of AWS infrastructure
![[Pasted image 20230314043632.png]]
## Characteristics
- Geographic separation - isolated fault domain
- Geopolitical Separation - different governance
- Location Control - performance

## Can be referred to via
- Region code
	- e.g. ap-southeast-2
- Region name
	- e.g. Asia Pacific (Sydney)test

# AWS Edge Locations
Local distribution points
Smaller than regions
Only have content distribution services
Have some type of edge computing
