An AWS account is a container for [[Identities]] and [[Resources]].

The account must be given an account name. For example, we can name an account `PROD`. Along with this account, an account root user is created, which has no restrictions at all withint he account.

Each AWS account requires a unique email address and a credit card.

You can create more [[Identities]] within an account that may have restrictions. This uses a service call [[Identity and Access Management]].
