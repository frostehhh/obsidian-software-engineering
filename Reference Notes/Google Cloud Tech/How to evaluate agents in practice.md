# Three-tier testing pyramid
- component-level unit testing
- trajectory and integration testing
- multi-agent end to end testing with human review
## Component-level unit testing
- Smallest unit of testing
- Examples
	- Evaluating if the right tool is called
	- Evaluating if the right skill is invoked
# Trajectory-Level OR Integration Tests
- Involves multi-step processes
- Examples
	- Is the constructed plan correct
	- Did the agent follow the expected plan and reasoning
	- Did the agent
## Multi-agent End to End Testing and Human Review
- Evaluating of a the entire agent system from the perspective of a human user
