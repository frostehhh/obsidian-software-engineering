- agent evaluation or testing is necessary to determine if our tool configuration
- AI and LLMs are probabilistic. Two separate executions may return different results
- Tests give us the grounds to evaluate the quality of our agent engineering
- This is equivalent to System Level Testing

# Agent Evaluation
The factors to evaluate involve but not limited to the following:
- produced output
- reasoning and planning
- tools used
	- Can it use the right tools for the right cases?
- memory and context
	- Does it retrieve information accurately?
	- Does it hallucinate?
- task success rate
