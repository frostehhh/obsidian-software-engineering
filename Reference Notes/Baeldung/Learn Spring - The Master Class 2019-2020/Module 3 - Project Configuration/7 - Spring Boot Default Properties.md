- Spring Boot has a lot of sensible defaults that are configured implicitly under the hood
- configurable via the application.properties file
- A few examples include **server.port**, **spring.main.banner-mode**, and there are a lot of other application properties that we can configure and we can refer to the Spring documentation for that.
