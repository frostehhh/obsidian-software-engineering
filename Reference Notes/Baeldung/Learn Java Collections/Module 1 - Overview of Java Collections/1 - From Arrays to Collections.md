- Arrays are fixed-size
	- Can be iterated via loops
- `String` is a subclass of `Object`. Therefore, you can assign a `String[]` to a variable `Object[]`. If a non-string type is assigned, for example an int, there will be a runtime error.
- Collections are a utility for manipulations of groups of objects. This provides common methods that can be used such as sorting, filtering, grouping, etc.
