# Benefits
- Improves readability
- less boilerplate code
- Clearer intent
- Utilize optimizations particularly for bulk operations
- Reduce chance of user error in case of using single modification operations as a part of a bulk operation
# Bulk operations
- addAll(Collection)
- addAll(index, Collection)
- removeAll(Collection)
- removeIf(Predicate)
- clear()
- replaceAll(Predicate)
