- Legacy Collections
	- Stack
	- Vector
	- Hashtable
- These collections are thread-safe legacy implementations where each already have modern equivalents

| Legacy    | Modern     |
| --------- | ---------- |
| Stack     | ArrayDeque |
| Vector    | ArrayList  |
| Hashtable | HashMap    |
