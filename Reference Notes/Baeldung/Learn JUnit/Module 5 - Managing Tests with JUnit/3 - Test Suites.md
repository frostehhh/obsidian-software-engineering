- A Test Suite is a way to group related tests together
# Annotations
- `@Suite`
	- mark a class as a test suite
	- must be paired with other annotations to indicate what tests are included
- `@SelectPackages`
- `@SelectMethods` and `@SelectMethod`
- `@SelectClasses`
- `@ExcludePackages`
- `@IncludeTags`
