In [[Cloud Computing]], we can have various types of cloud environments.

### Public
Available to the public. A single use of 1 public cloud
For example, AWS, Azure, and Google

### Multi Cloud
Using multiple [[Public vs Private vs Hybrid vs Multi Cloud#Public|public]] cloud environments. For example, one part of your system uses [[Public vs Private vs Hybrid vs Multi Cloud#Public|public]] AWS and the other uses Azure

### Private Cloud
Dedicated to businesses on-premises *real* cloud. For example, AWS outposts, Azure Stack, and Anthos. 

### Hybrid
A combination of [[Public vs Private vs Hybrid vs Multi Cloud#Public|public]] and [[Public vs Private vs Hybrid vs Multi Cloud#Private Cloud|private]] cloud environments
> [!important] 
>  Hybrid cloud is not public cloud + legacy on-premises. This is a hybrid environment or hybrid network.
