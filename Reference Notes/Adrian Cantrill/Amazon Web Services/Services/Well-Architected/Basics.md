# Areas to consider for serverless workload
- **Compute Layer** - managing requests for external sources
- **Data Layer** - persistent storage
- **Messaging and streaming layer** - communication between components
- **User management and identity layer**
- **Edge layer** - manages presentation layer and connectivity to external customers
- **Systems monitoring and deployment** - system visibility through metrics and creates contextual awareness of how it operates and behaves over time
- **Deployment approaches**
- **Lambda version control**
