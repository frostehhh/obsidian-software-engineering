- designed to overcome IPv4 shortages. For IPv4 only. Doesn't make sense for IPv6 
- Translates Private IPv4 address to Public
- Types of NAT
	- [[Static NAT]] - 1 private to 1 fixed public address (Used by AWS Internet Gateway - IGW)
	- [[Dynamic NAT]] - 1 private to 1st available Public
	- [[Port Address Translation]] (PAT) - many private to 1 public (NATGW)
