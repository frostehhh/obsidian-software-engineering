# What is Spring?
Spring is a framework used to quickly create enterprise Java applications. [^1]

# Spring Inversion of Control (IoC) Container
Inversion of control is a principle where in Spring context, Spring creates or instantiates the dependencies that an object needs. For example, we have a class and in the constructor, dependencies are defined. Spring initializes those dependencies for that class's use.[^2]

- The `org.springframework.context.ApplicationContext` interface represents the Spring IoC container
