- Spring is a Dependency Inversion framework that uses the inversion of control principle
- Spring Boot is a wrapper over Spring that simplifies Spring usage
	- has sensible default to reduce boilerplate

# Components of Spring and SpringBoot
## Annotations
- SpringBootApplication
- Configuration - class annotation indicating that there are beans in this class
- Bean - method annotation. Fundamental functional unit in Spring
