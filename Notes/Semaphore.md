- Means of access control by multiple processes in a system to a common resource
- Used for process synchronization -> specified limited number of processes can access a shared resource at a time
# Types
1. counting
	0 to N processes can access at a time
2. binary
	- 0 to 1 only
	- works similarly to a lock
