- JEE is an enterprise platform consisting of sets of tools and APIs built for large-scale architecture and development
- Multi-tier architecture consisting of
	- client tier
	- presentation tier
	- business tier
	- database tier

# Tools and Concepts(not all)
## Servlets
- Extends web servers, processes client requests
## Java Server Pages
- HTML with embedded Java code
## Java Server Faces

## Java EE vs Jakarta EE
- Java EE is the initial solution created.
- Jakarta EE is an alternative developed by the community under the Eclipse Foundation, which is continued today.
