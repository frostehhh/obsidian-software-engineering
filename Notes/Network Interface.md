A network interface refers to the physical or virtual point of connection used to communicate with other devices.[^1] An example of a virtual network interface is the loopback(127.0.0.1). It can also be a simulation of a network interface.

A physical network interface is a network interface controller[^3]

Each network interface must have its own unique IP Address[^2]
