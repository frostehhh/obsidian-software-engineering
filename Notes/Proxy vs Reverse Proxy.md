# Proxy
Middleman for clients to communicate with servers
## Purposes
- Anonymity
- Caching
- Logging and Monitoring
- Blocking
- Microservices

# Reverse Proxy
Middleman for server communication

## Purposes
- Caching
- Microservices
- Load balancing
- Ingress
- Canary deployment
