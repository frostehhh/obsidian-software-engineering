- Two-way conversion of plain text to a ciphertext. 
- Ciphertext can be converted back to plain text
- It is like hiding data in plain sight

###### Encryption Concepts
1. Plaintext
2. Algorithm
3. Key
4. Ciphertext

> [!info] 
> Plaintext is taken by an encryption algorithm using a key to translate the plaintext into ciphertext.
> Decryption is the reverse order of the encryption concepts.
