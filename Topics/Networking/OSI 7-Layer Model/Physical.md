- Physical shared medium
- Standards for transmitting onto the medium
- Standards for receiving from the medium
- No access control
- No uniquely identified devices
- No device => device communications


#### Example
There are 2 laptops that are connected by a network cable between the network interface cards contained by each laptop.

The network cable can be copper(electrical), fibre(light) or WIFI(RF)

![[Pasted image 20230102032304.png]]

The limitation of this example is that there can be a network of only 2 devices. In another example, we can have a physical network hub that allows 4 network cables to be connected to it.

> [!note] 
> Based on the previous example, by default, data is broadcasted. Meaning, data is sent to all other devices.
> 
> If multiple devices transmit at once, a collision occurs.
> 
> L1 has no media access control and no collision detection.
