Uniquely references a resource in any AWS accounts. Can reference group of resources via wildcards

# Supported Formats
```
arn:`partition`:`service`:`region`:`account-id`:`resource-id` arn:`partition`:`service`:`region`:`account-id`:`resource-type`/`resource-id` arn:`partition`:`service`:`region`:`account-id`:`resource-type`:`resource-id`
```
> [!note] 
> Some resources do not require a region parameter or require a wildcard
